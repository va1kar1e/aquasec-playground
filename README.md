# aquasec-playground

Playground for Testing Aquasec Cloud Security APIs

## Environment File

Create file `.env` in root directory

```plaintext
# Aquasec Credentials
AQUASEC_USER="<USERNAME>"
AQUASEC_PASSWORD="<PASSWORD>"

# Aquasec URL
AQUASEC_NONPROD_URL = "<URL1>"
AQUASEC_PROD_URL = "<URL2>"
```
